VSCode 使用指南

掌握Visual Studio Code，提升开发效率

VSCode是现代开发者的首选编辑器，让我们通过实际例子来学习

案例教学

以下是一些实际的VSCode使用技巧和功能演示：


自动换行Word Wrap

vscode中一行文本太长，如何不使用左右滚轮，而是转到下一行

Mac 上，快捷键是： Option + Z

按下 Cmd + Shift + P 打开命令面板，输入 word wrap 选择 View: Toggle Word Wrap (视图: 切换自动换行)

智能提示IntelliSense

代码自动补全功能

自动补全示例：

输入 doc → 自动提示 document

输入 getEl → 自动提示 getElementById

输入 addEv → 自动提示 addEventListener

文件管理File Management

高效的文件操作

常用快捷键：

• Ctrl + P - 快速打开文件

• Ctrl + Shift + P - 命令面板

• Ctrl + B - 切换侧边栏

• Ctrl + ` - 切换终端

💡 提示：使用 Ctrl + P 快速搜索文件

插件推荐Extension Recommendations

必备的VSCode插件

Live Server - 本地服务器
Prettier - 代码格式化
ESLint - 代码检查
Auto Rename Tag - 自动重命名标签

调试功能Debugging

强大的调试工具

调试步骤：
  1. 在代码行设置断点
  2. 按 F5 开始调试
  3. 使用调试控制台
  4. 监视变量值变化

代码片段Code Snippets

自定义代码模板

常用片段：

html → HTML5 基础结构

css → CSS 样式模板

js → JavaScript 函数模板

输入片段名称后按 Tab 键展开

Git集成Git Integration

版本控制功能

Git 操作：

• Ctrl + Shift + G - 打开源代码管理

• Ctrl + K, Ctrl + S - 暂存更改

• Ctrl + Enter - 提交更改

📝 左侧面板显示文件修改状态

主题定制Theme Customization

个性化界面设置

推荐主题：
Dark+
Light+

按 Ctrl + K, Ctrl + T 切换主题

工作区设置Workspace Settings

项目配置管理

设置文件：
.vscode/settings.json - 工作区设置
.vscode/launch.json - 调试配置
.vscode/tasks.json - 任务配置

终端集成Integrated Terminal

内置终端功能

终端操作：

• Ctrl + ` - 打开/关闭终端

• Ctrl + Shift + ` - 新建终端

• Ctrl + Shift + C - 复制终端内容

$ npm install
$ git status
$ code .
