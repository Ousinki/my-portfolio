Linux 基础教程

学习Linux命令行，掌握开发必备技能

Linux是开发者的重要工具，让我们通过实际例子来学习

案例教学

以下是一些实际的Linux命令示例和交互演示：


文件操作File Operations

基本的文件和目录操作命令

$ ls -la # 列出所有文件详细信息
$ cd /home/user # 切换目录
$ pwd # 显示当前目录
$ mkdir mydir # 创建目录
$ touch file.txt # 创建文件

文本处理Text Processing

查看和编辑文本文件

$ cat file.txt # 显示文件内容
$ head -n 10 file.txt # 显示前10行
$ tail -n 10 file.txt # 显示后10行
$ grep "keyword" file.txt # 搜索关键词

💡 grep 命令是文本搜索的强大工具

系统信息System Info

查看系统状态和进程

$ ps aux # 查看所有进程
$ top # 实时查看系统状态
$ df -h # 查看磁盘使用情况
$ free -h # 查看内存使用情况

权限管理Permission Management

文件和目录权限设置

$ chmod 755 file.txt # 设置文件权限
$ chmod +x script.sh # 添加执行权限
$ sudo command # 以管理员身份执行
$ su - username # 切换用户

⚠️ sudo 命令需要谨慎使用，确保了解命令的作用

网络操作Network Operations

网络相关命令

$ ping google.com # 测试网络连接
$ wget url # 下载文件
$ curl -I url # 获取HTTP头信息
$ ssh user@host # SSH连接

包管理Package Management

软件包安装和管理

Ubuntu/Debian:
$ sudo apt update # 更新包列表
$ sudo apt install package # 安装软件包
$ sudo apt remove package # 卸载软件包
CentOS/RHEL:
$ sudo yum install package # 安装软件包
$ sudo yum remove package # 卸载软件包

压缩解压Compression

文件压缩和解压缩

$ tar -czf archive.tar.gz files/ # 创建压缩包
$ tar -xzf archive.tar.gz # 解压缩包
$ zip -r archive.zip files/ # 创建zip压缩包
$ unzip archive.zip # 解压zip文件

📦 tar 命令是Linux中最常用的压缩工具

环境变量Environment Variables

设置和使用环境变量

$ echo $PATH # 查看PATH环境变量
$ export VAR=value # 设置环境变量
$ env # 查看所有环境变量
$ alias ll='ls -la' # 创建命令别名

管道和重定向Pipes & Redirection

命令组合和数据流控制

$ ls -la | grep txt # 管道操作
$ command > file.txt # 输出重定向到文件
$ command >> file.txt # 追加输出到文件
$ command < input.txt # 从文件读取输入

🔗 管道 | 可以将一个命令的输出作为另一个命令的输入

实用技巧Practical Tips

提高效率的Linux技巧

快捷键：
  • Ctrl + C - 中断当前命令
  • Ctrl + D - 退出终端
  • Ctrl + L - 清屏
  • Tab - 命令补全
  • ↑/↓ - 历史命令

脚本编程Shell Scripting

简单的Shell脚本示例

#!/bin/bash
echo "Hello, World!"
echo "Current date: $(date)"
echo "Current user: $(whoami)"

保存为 .sh 文件，使用 chmod +x 添加执行权限

代码示例Code Example

完整的Shell脚本

#!/bin/bash
# 系统信息脚本
echo "=== 系统信息 ==="
echo "主机名: $(hostname)"
echo "操作系统: $(uname -s)"
echo "内核版本: $(uname -r)"
echo "当前用户: $(whoami)"
echo "当前目录: $(pwd)"
echo "磁盘使用: $(df -h / | tail -1)"
echo "内存使用: $(free -h | grep Mem)"
